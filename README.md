# Ribbons::Rails

Rails assets and view helpers for 3D ribbons.

## Installation

Add this line to your application's Gemfile:

    gem 'ribbons-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ribbons-rails
